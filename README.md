# 🌊 VectorFlow AI

VectorFlow is a professional Enterprise RAG (Retrieval-Augmented Generation) application. It acts as a strict, focused AI assistant that answers questions based exclusively on internal company documents.

## 🛠️ Built With
* **Frontend:** Streamlit
* **AI Model:** Google Gemini (2.5-Flash & Embedding-2)
* **Framework:** LangChain
* **Vector Database:** ChromaDB
